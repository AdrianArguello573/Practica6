¿Qué es Frontend?

El frontend es la parte de una aplicación o sitio web que interactúa directamente con los usuarios. Involucra el diseño visual, la disposición de los elementos y la experiencia del usuario.

Fundamentos del Frontend

  • HTML: Define la estructura del contenido.
  • CSS: Estiliza y define la apariencia visual.
  • JavaScript: Agrega interactividad y dinamismo.
